# Media Labeler v3.1

Pemisah media video dan poto. Buat misah-misahin file doang biar rapih.

### Cara Pakai:
1. Install dulu bahan-bahannya: `pip install -r requirements.txt`
2. Jalanin: `python labeler_app.py`
3. Pilih folder asal sama folder tujuan.
4. Klik tombol kategorinya buat mindahin file.
